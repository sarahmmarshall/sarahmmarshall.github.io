SceneManager.cpp – Project One

Course: CS330 - Computer Graphics and Visualization

Original | Enhanced

Rendered 3D objects in C++ and OpenGL. Enhanced scene with new meshes, textures, and lighting.

This project is SceneManager.cpp, which comes from CS330 - Computer Graphic and Visualization. The purpose of this code is to render 3D objects in a scene in the C++ programming language and OpenGL. This program consists of many classes that all work together to render 3D objects, apply textures to meshes, apply light sources to the scene, and move around the scene with a camera using keys and the mouse. The specific components in the project that showcase my skills and abilities in software development is the addition of more meshes(objects) in the 3D scene and rendering them with textures, enhancing the textures on past objects to make the scene look more realistic, and enhancing the lighting of the scene. The enhancements improved the project as it made it more advanced with better textures, more objects, and better lighting. The enhancements made the 3D scene more realistic.

ContactService.java – Project Two

Course: CS320 - Software Test, Automation & QA

Original | Enhanced

Improved performance by replacing an ArrayList with a HashMap for managing contacts in Java. Verified efficiency with JUnit.

This project is ContactService.java, which comes from CS320 - Software Test, Automation & QA. The purpose of this code is to store contacts using a data structure and to be able to add, delete, retrieve, and update contacts in the java programming language. This program consists of four classes, the two classes Contact.java and ContactService.java work together to add, delete, retrieve, and update contacts. The other two classes ContactTest.java and ContactServiceTest.java are test classes to run JUnit tests to verify the other two classes work as intended. The specific components in the artifact that showcase my skills and abilities in algorithms and data structures is the replacement of a data structure with a better suited one. The enhancements improved the project as it made it more advanced with a better data structure. We can see in the JUnit test that the program preformed quicker with a HashMap than with an array list data structure.
